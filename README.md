# 📚 儿童学习提醒助手

一个帮助孩子养成健康学习习惯的 Web 应用。支持手动计时和智能摄像头检测两种模式。

## ✨ 功能特点

### V1.0 - 基础定时提醒
- ⏱️ **学习计时**：精确记录学习时长
- 🔔 **定时提醒**：到达设定时间自动弹窗提醒休息
- 🔊 **声音提示**：提醒时播放提示音
- ⚙️ **自定义设置**：可调整提醒间隔（1-120分钟）
- 💾 **配置保存**：设置自动保存到本地

### V2.0 - 智能摄像头检测
- 📷 **自动检测**：摄像头检测孩子坐下后自动开始计时
- 👀 **走神提醒**：检测到孩子走神时自动提醒
- 🚫 **自动暂停**：孩子离开座位时自动暂停计时
- 🎯 **双模式**：支持手动模式和智能模式切换

## 📖 使用方法

### 手动模式（V1.0）
1. 点击"开始学习"按钮开始计时
2. 学习中途可以暂停或重置
3. 到达设定时间后会弹出休息提醒
4. 点击"我知道了"后重新开始计时

### 智能模式（V2.0）
1. 勾选"📷 智能模式"开关
2. 允许浏览器访问摄像头
3. 坐到摄像头前，系统自动开始计时
4. 离开座位时自动暂停
5. 走神超过5秒会收到提醒

### 设置
- 点击"⚙️ 设置"按钮打开设置面板
- 可调整提醒间隔（默认25分钟）
- 可开关提示音
- 可调整走神检测阈值
