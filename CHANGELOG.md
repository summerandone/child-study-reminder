# 更新日志

所有重要的项目变更都会记录在此文件中。

## [2.0.0] - 2024-12-28

### 新增
- 📷 智能摄像头检测模式
  - 自动检测孩子是否坐在座位上
  - 坐下自动开始计时
  - 离开自动暂停计时
- 👀 走神检测功能
  - 检测头部是否偏离屏幕
  - 走神超过阈值自动提醒
  - 恢复专注后自动关闭提醒
- 🔄 模式切换
  - 支持手动模式和智能模式
  - 一键切换，灵活使用
- ⚙️ 新增设置项
  - 走神检测阈值（3-30秒）

### 技术
- 使用 MediaDevices API 获取摄像头
- 简单的图像分析算法（肤色检测）
- 无需复杂的机器学习库

---

## [1.0.0] - 2024-12-27

### 新增
- ⏱️ 学习计时功能
  - 开始/暂停/重置计时
  - 实时显示学习时长
- 🔔 定时休息提醒
  - 到达设定时间自动弹窗
  - 提醒喝水、看远方
- 🔊 提示音功能
  - 提醒时播放声音
  - 可在设置中开关
- ⚙️ 设置功能
  - 自定义提醒间隔（1-120分钟）
  - 配置自动保存到本地
- 📱 响应式设计
  - 支持桌面和移动设备

### 技术
- 纯前端实现（HTML + CSS + JavaScript）
- 模块化代码结构
- localStorage 配置持久化
- Web Audio API 生成提示音
